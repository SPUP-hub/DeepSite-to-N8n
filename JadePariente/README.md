# JadePariente

Ce dossier regroupe les **documents, visuels et fichiers médias** liés au projet de Jade Pariente.

Il contient notamment :
- Le **CV** et le **portfolio** en PDF
- Des visuels optimisés pour le web
- Les assets nécessaires à la mise en ligne ou au traitement automatisé dans DeepSite/N8n

Ce répertoire est destiné à une **utilisation modulaire** dans des projets créatifs ou de démonstration.
